## 🛠️ Project Setup

1. Clone the repo  
   `git clone git@github.com:thelab33/ConnectAtxElite.git`
2. Install Python deps  
   `pip install -r requirements.txt`
3. Run it  
   `flask run`
4. Visit [localhost:5000](http://localhost:5000)
